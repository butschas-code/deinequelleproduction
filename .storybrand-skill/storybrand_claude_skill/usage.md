How to use

1) Paste the entire content of the current website (all pages if possible) below the prompt.
   - Include navigation labels, headings, service lists, addresses, phone numbers, certifications, and reviews.

2) Optional: Add constraints:
   - “Keep these page titles”
   - “Prefer short German headings”
   - “Primary CTA should be Offerte anfragen”

3) The model will output:
   A) Source Map
   B) BrandScript (SB7)
   C) Homepage wireframe script
   D) QA checklist

Tip: If you provide Google reviews separately, paste them as a “REVIEWS” section at the end.
