StoryBrand Euro Homepage Skill (Claude / Cursor)

Files:
- prompt.md: The complete master prompt (paste into Claude as a “skill” / system prompt, or into Cursor).
- schema.org.template.json: A JSON-LD template for reviews/business markup.
- usage.md: How to use this with website text.
- checklist.md: QA checklist you can run on outputs.

This skill is designed to:
- Extract only what’s in the provided website text
- Build a StoryBrand BrandScript (SB7)
- Produce a top-to-bottom homepage script with calm European tone
- Embed reviews in schema.org JSON-LD
