# StoryBrand Euro Homepage Skill

## What this is
A copywriting + information architecture skill based on Donald Miller’s StoryBrand framework, adapted for European sensibility:
- calm, confident, understated tone
- opportunity framing first (aspiration + clarity)
- fifth-grade reading level
- strict source-accuracy (no invented claims)
- BrandScript (SB7) + full homepage wireframe script
- reviews embedded with schema.org JSON-LD

## How to use
1. Set **prompt.md** as the skill/system prompt in Claude (or paste it into Cursor).
2. Paste the complete website text (all pages) into the chat.
3. Optionally add constraints (preferred CTA label, keep certain headings, etc.).
4. The model must output in order:
   A) Source Map  
   B) BrandScript (SB7)  
   C) Homepage wireframe script  
   D) QA checklist  

## Files
- **SKILL.md**: overview + required metadata (this file)
- **prompt.md**: the actual master prompt
- **usage.md**: quick usage guide
- **checklist.md**: QA checklist
- **schema.org.template.json**: JSON-LD template for reviews

## Guardrails
- Every fact must come from the provided website text.
- Missing info must be marked as **[Missing: …]**.
- All services/offers must be included (none dropped).
